This tool allows you to:

* Select the subdomain discovery tools you used
* Paste raw results from multiple tools

* Remove duplicates
* Get a sorted list of unique subdomains
* Copy results instantly

## Features

* Target domain filtering
* Automatic duplicate removal
* Sorted unique output
* Total subdomain counter
* Copy-to-clipboard functionality
* Clean dark UI with Tailwind CSS
* Fully client-side (no backend required)

---

## How It Works

1. Enter your target domain (example.com).
2. Select the tools you used (optional tracking).
3. Paste all raw enumeration results into the input box.
4. Click **Generate Unique Subdomains**.
5. Copy or export the cleaned results.

The app filters entries that:

* Match the target domain
* End with `.targetdomain.com`
* Are valid subdomain entries
* Removes duplicates automatically
